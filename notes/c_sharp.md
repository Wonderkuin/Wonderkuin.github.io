#### 风格

```
PascalCase
    ComponentModel
    Configuration
    HttpFileCollection
camelCase
    quotient
    firstName
    ioStream
    theDreadPirateRoberts

更重视清晰 而不是简短
不要使用单词缩写
不要使用不广泛接受的词，即使被广泛接受，非必要也不要用

下划线虽然合法，但标识符一般不要包含下划线，连字符，其他非字母数字字符
不使用匈牙利命名法

极少数情况下，有的标识符有特殊含义，比如Main


要把两个字母的首字母全部大写，除非是camelCase的第一个单词
包含三个或更多字母的首字母缩写词，仅仅第一个字母大写，除非是camelCase的第一个单词
camelCase开头首字母缩写词中， 所有字母都不大写
不要使用匈牙利命名法

@可作为变量名使用，例如@return @throw
微软实现中， 还有四个未文档化的保留字
__arglist
__makeref
__reftype
__refvalue
仅在罕见的互操作情形下使用，平时可忽略
安全起见，自己不要创建这样的标识符
```
