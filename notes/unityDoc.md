# Unity 文档记录

---

```
Project窗口:
Project窗口 单击时按住alt键，递归方式展开或折叠所有
    （小三角图标）
Favorites 可以将所需项拖动过去，也可以保存查询结果
    试了，可以保存查询结果，但是怎么保存单个选择？
Search bar
    t: 资源筛选
    l: 标签筛选
    在选中文件夹搜索：搜索后，有一个search标签，可以选择
        All InPackages InAssets 当前目录 AssetStore

Scene窗口:
场景视图辅助图标 场景XYZcube
    右键 可以展开菜单
    Shift + 单击 可以返回透视图
    mac触控板可以触控操作
    按住alt，鼠标左右键可以旋转相机/缩放视图
    fps模式：右键+wasd移动，shift加速
    scene窗口，摄像机图标内，可以修改摄像机速度
    移动快捷键 快速移动 alt+鼠标中键，拖动
    移动快捷键 旋转 alt+鼠标右键，拖动
    飞跃模式更改速度 鼠标滚轮
    F 定位到游戏对象
    Shift+F 锁定到游戏对象，即使对象移动
拾取和选择游戏对象
    edit preferences修改表现颜色
    gizmos修改轮廓和线框
    选择游戏对象
        单击，支持shift和ctrl
    拾取设置
        hierarchy设置可拾取，可见
游戏对象定位
    辅助图标控制柄
        位置：Pivot/Center
        Pivot将辅助图标定位在游戏对象的实际轴心点，由变换组件进行定义
        Center将辅助图标定位在中心位置
        旋转：Local/Global
        Local保持辅助图标相对于游戏对象旋转
        Global将辅助图标固定在世界空间方向上
    对齐
        世界网格对齐：对齐到xyz轴投影的网格，或将游戏对象沿xyz轴进行递增变换。
            仅当使用world或者global控制柄方向时，这种对齐方式才可用。
        表面对齐：将游戏对象对齐到任何碰撞体的交点
        顶点对齐：将任何顶点从给定网格对齐到另一网格的顶点或表面的位置。
            可将顶点对齐到顶点，将顶点对齐到表面，将轴心对齐到顶点。
    
```
