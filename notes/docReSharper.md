# ReSharper IDEA 快捷键记录

---

```
Ctrl + D 复制上一行
    Ctrl + E 显示最近编辑的文件
    Shift + Alt + L 定位资源管理器位置
  Alt + F7 查找引用
  Ctrl + N GotoEverything
  Ctrl + Shift + N GotoFile
  Ctrl + F12 当前类查找
  F2 重命名
  Ctrl + Tab 活动文件替换
  Ctrl + Shift + Alt + 向上/向下 上下行代码交换位置
  Ctrl + W 快速选中整个/一块单词
  Ctrl + Shift + W
  Ctrl + Alt + F Clean Code
  Ctrl + Alt + J sorround with template 快速添加语句块 如 if for try catch using #region
  Ctrl + Q 快速文档
  Alt + F12 显示下一个Error
  Ctrl + 点击 跳转到声明
  Ctrl + Shift + G 检查可用操作
  Ctrl + Shift + Space 智能补全
  Alt + Enter
  Ctrl + F11 类结构 对象浏览器
  Ctrl + Shift + R 重构
  Alt + Insert 自动生成代码
  Ctrl + Alt + J 包围代码
```
